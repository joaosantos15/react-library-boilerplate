
## TL;DR

A simple boilerplate for React Components Libraries

- ✅ Typescript
- ✅ Rollup (bundles images and JSON files as well)
- ✅ SASS


## Get Started

```sh
> npm install
> npm run build
```


Based on this amazing guide: https://blog.harveydelaney.com/creating-your-own-react-component-library/
